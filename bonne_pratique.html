Tests automatisés¶

Test automatisé verfiant que le code produit les bons résultats. Le but principal n’est pas de prouver que le code est bon, mais de s’assurer qu’aucunes modifications de ce dernier ne viennent casser des fonctionnalités déjà validées. Ils permettent aussi à un utilisateur du code de s’assurer que ce dernier est bien installé et fonctionne correctement sur sa machine.

Pour des code Python, le logicel PyTest peut être utilisé.
